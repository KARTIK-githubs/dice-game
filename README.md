# 🎲 Dice Game

A simple dice game built using HTML, CSS, and JavaScript.

This project was created while learning JavaScript DOM manipulation. Clicking the **Roll Dice** button generates random values for both players, updates the dice images, and displays the winner.

## Features

* Random dice generation
* Roll Dice button interaction
* Dynamic image updates
* Winner and draw detection
* DOM manipulation using vanilla JavaScript

## What I Learned

Building this project helped me understand:

* How JavaScript interacts with HTML elements
* DOM selection using `querySelector()`
* Event handling with `addEventListener()`
* Updating attributes using `setAttribute()`
* Changing content dynamically using `textContent`
* Using `Math.random()` to generate random outcomes

## How the Game Works

1. Click the **Roll Dice** button.
2. Both players get a random number between 1 and 6.
3. The dice images are updated based on the generated numbers.
4. The player with the higher number wins.
5. If both numbers are equal, the game ends in a draw.

## Why I Built This

I built this project as part of my web development learning journey. The goal was not just to create a dice game, but to get comfortable with JavaScript fundamentals and understand how the DOM can be manipulated to create interactive web applications.
